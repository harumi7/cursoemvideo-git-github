Git and GitHub: The full guide

Branch

In a project, different branches are used to work on specific tasks, such as features, design, and content.

It works like a tree with many branches! Once a branch is complete, you can merge it into the master (main) branch.

Fork

A "fork" on GitHub is a copy of a repository that allows you to make changes to the code without affecting the original repository. It's very useful for collaboration and open-source projects development.

Issue

On GitHub, an "issue" is a tool for tracking tasks, enhancements, and bugs for your projects. Issues can be used to discuss and plan new features, report problems, and track the progress of different parts of a project.
